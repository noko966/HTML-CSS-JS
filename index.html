What is HTML?

HTML stands for HyperText Markup Language.

It's the foundational language for creating and structuring content on the web.

HTML tells web browsers how to display a webpage's text, images, and other forms of multimedia.

HTML language, not a programming language.

Basic Anatomy of an HTML Tag

Opening Tag: This is the first part of an HTML element. It consists of the element's name enclosed within angle brackets. For example, <p> is the opening tag for a paragraph element.

Content: This is the part that goes between the opening and closing tags. It can include text, images, links, other HTML elements, or even be empty for some elements. For example, in < p > Hello, World! </ p > , "Hello, World!" is the content.

Closing Tag: This marks the end of the element. It is similar to the opening tag but includes a forward slash before the element name. For example, </ p > is the closing tag for a paragraph element.

Attributes: These provide additional information about the element, usually in the form of a name-value pair, and are included in the opening tag. They can specify styles, classes, IDs, data properties, and more...
